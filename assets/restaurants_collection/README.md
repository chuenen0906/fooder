## 照片收集來源建議

1. **Google Maps**：搜尋店家名稱，查看店家照片
2. **店家官網**：如果有官方網站，通常有高品質照片
3. **Facebook/Instagram**：店家社群媒體帳號
4. **美食部落格**：搜尋店家名稱 + 食記
5. **實地拍攝**：親自前往店家拍攝（最佳選擇）

## 照片要求

- **尺寸**：建議 800x600 或 1200x800
- **格式**：JPG 或 PNG
- **品質**：清晰、光線充足
- **內容**：突出店家特色和美食

## 注意事項

- 確保照片版權合法
- 避免使用有浮水印的照片
- 照片要能代表店家特色
- 建議每間店家至少 2-3 張照片
